# Анализ влияния параметров сети на ошибку локализации методом линейной регрессии

## Описание проекта

В данном проекте проводится статистический анализ набора данных моделирования беспроводной сенсорной сети и исследуется влияние параметров сети на среднюю ошибку локализации (`ale`) с использованием методов линейной регрессии.

Цель проекта:

* исследовать зависимости между параметрами сети и ошибкой локализации
* построить модели простой линейной регрессии
* построить модель множественной линейной регрессии
* оценить статистическую значимость признаков
* проверить мультиколлинеарность факторов
* сравнить модели с помощью метрик качества

---

## Используемые данные

Файл данных:

```
mcs_ds_edited_iter_shuffled.csv
```

Описание признаков:

| Признак      | Описание                                        |
| ------------ | ----------------------------------------------- |
| anchor_ratio | доля опорных узлов                              |
| trans_range  | радиус передачи                                 |
| node_density | плотность узлов                                 |
| iterations   | число итераций алгоритма                        |
| sd_ale       | стандартное отклонение ошибки локализации       |
| ale          | средняя ошибка локализации (целевая переменная) |

Целевая переменная:

```
ale
```

## Этапы анализа

### 1. Загрузка данных

Данные загружаются из CSV-файла в DataFrame:

```python
df = pd.read_csv('mcs_ds_edited_iter_shuffled.csv')
```

---

### 2. Визуализация зависимостей

Построены столбчатые диаграммы зависимости целевой переменной `ale` от параметров:

* anchor_ratio
* trans_range
* node_density
* iterations

Это позволило предварительно оценить характер зависимостей.

---

### 3. Корреляционный анализ

Рассчитана матрица корреляций:

```python
df.corr()
```

Построена тепловая карта корреляций с использованием seaborn.

Основные выводы:

* сильная отрицательная корреляция между `node_density` и `ale`
* умеренная отрицательная корреляция между `iterations` и `ale`
* сильная положительная корреляция между `sd_ale` и `ale`

---

### 4. Построение моделей простой линейной регрессии

Построены отдельные модели регрессии:

```
anchor_ratio → ale
trans_range → ale
node_density → ale
iterations → ale
```

Для каждой модели вычислены:

```
intercept
slope
```

и построены графики зависимости.

---

### 5. Построение модели множественной линейной регрессии

Построена модель:

```
ale ~ anchor_ratio + trans_range + node_density + iterations + sd_ale
```

Результаты:

```
R² на обучающей выборке: 0.7278
R² на тестовой выборке: 0.6717
```

Это означает, что модель объясняет около 67% дисперсии ошибки локализации.

---

### 6. Проверка мультиколлинеарности (VIF)

Рассчитан коэффициент инфляции дисперсии:

```
variance_inflation_factor()
```

Результаты показали:

* критической мультиколлинеарности нет
* признаки пригодны для использования в модели

---

### 7. Проверка статистической значимости коэффициентов

Использована модель OLS из библиотеки statsmodels:

```python
model = sm.OLS(y_train, X_incl_const)
```

Получены:

* коэффициенты регрессии
* p-value
* R²
* BIC

Статистически значимые признаки:

```
node_density
iterations
sd_ale
```

Менее значимые признаки:

```
anchor_ratio
trans_range
```

---

### 8. Сравнение моделей по критерию BIC

Сравнивались две модели:

Полная модель:

```
BIC = 9.76
R² = 0.7278
```

Сокращённая модель:

```
BIC = 30.34
R² = 0.5942
```

Вывод:

полная модель показывает лучшее качество.

---

### 9. Оценка качества модели

Для оценки качества модели использованы метрики:

```
MAE — средняя абсолютная ошибка
MSE — среднеквадратичная ошибка
R² — коэффициент детерминации
BIC — информационный критерий Байеса
VIF — проверка мультиколлинеарности
p-value — статистическая значимость коэффициентов
```
